Captulo 3

  • Funo scale: altera o formato da escala do eixo (ex: de decimal para logartmico).
    • labels: altera o formato do texto da escala do eixo (biblioteca scales). colocada dentro do scale.
  • Parmetro alpha: altera a transparncia do objeto (0 transparente e 1 opaco).
  • labs: camada que altera os textos do grfico (x, y, title, subtitle e caption).
  • Parmetro fill: altera a cor de reas, como sombras de linhas de regresso e barras.
  • Funo log: configura as cores de um elemento para um gradiente em funo de uma varivel.
  • fig.width e fig.height: alteram o tamanho das imagens geradas no RMarkdown, devendo ser colocadas como uma funo default no primeiro bloco de comando (knitr::opts_chunk$set(fig.width = 8, fig.height = 5)) ou nas configuraes de um bloco individual dentro das chaves.
  • ggsave: Salva a ltima imagem gerada pelo terminal.
    • plot: Salva um grfico salvo em uma varivel.
  • Biblioteca here: possibilita configurar onde uma imagem ser salva.

Captulo 8

  • subset: Filtra uma base de dados.
  • geom_text_repel: Legenda os pontos do grfico. Precisa da biblioteca ggrepel.
  • theme: Pode alterar a posio da legenda.
  • RColorBrewer: Biblioteca de paletas de cores.
    • scale_color_brewer para adicionar a paleta como uma camada.
    • possvel criar uma paleta fazendo uma lista com os cdigos das cores e usando a funo scale_color_manual.
    • brewer.pal: Pega uma paleta padro do R.
  • theme_set: Define um tema para todos os grficos subsequentes.
    • possvel configurar o tema de um grfico individualmente adicionando uma camada theme_().
    • ggthemes: Pacote de temas.
